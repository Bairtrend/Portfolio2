Méthode MVC

Le modèle MVC (Modèle-Vue-Contrôleur) divise une application en trois composants : le Modèle qui gère les données, la Vue qui affiche l'interface utilisateur, et le Contrôleur qui fait le lien entre les deux en traitant les actions de l'utilisateur.
